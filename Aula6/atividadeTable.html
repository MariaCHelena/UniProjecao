Contribuição para o perfil do egresso
A disciplina apresenta ao aluno os principais conceitos, métodos e técnicas de interação com a Web. Estes conceitos são importantes para que o egresso perceba as necessidades interativas dos usuários com sistemas computacionais, facilitando a geração de interfaces que proporcionem interações eficazes, eficientes e agradáveis. O futuro profissional da área de sistemas de informação adiquirá, com esta disciplina, os conhecimetos necessários para desenvolver interfaces para o ambiente Web, devidamente padronizados.
