CONTRIBUIÇÃO PARA O PERFIL DO EGRESSO

A disciplina apresenta ao aluno os principais conceitos, métodos e técnicas de interação com a Web. Estes conceitos são importantes para que o egresso perceba as necessidades interativas dos usuários com sistemas computacionais, facilitando a geração de interfaces que proporcionem interaçoes eficazes, eficientes e agradáveis. O futuro profissional da área de sistemas de informação adquirirá, com esta disciplina, os conhecimentos necessários para desenvolver interfaces para o ambiente Web, devidamente padronizados.

CONTEÚDO PROGRAMÁTICO

  1. Unidade - Histórico da Internet e Introdução aos conceitos básicos de Internet
    1. Histórico da Internet;
    2. Serviços WWW, E-MAIL, FTP(Conceituação de protocolos HTTP e TCP/IP)
    3. Conceituação e definição de sites na WEB
    4. Tipos de Linguagem de Programação
    5. Conceituação de Cliente e Servidor
  2. Unidade - Linguagem HTML
    1. Fundamentos de HTML
    2. Atributos de texto, inserção de imagens, links
    3. Componentes HTML
